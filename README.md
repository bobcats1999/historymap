# 西方历史大网

这是一个静态前端历史学习地图。页面需要通过本地静态服务访问，服务进程停止后 `127.0.0.1:8011` 就会失效。

## 启动

```bash
npm run dev
```

然后打开：

```text
http://127.0.0.1:8011
```

如果页面无法访问，先确认运行 `npm run dev` 的终端没有关闭。启动脚本会固定绑定 `127.0.0.1`，避免系统默认使用 IPv6 导致 `127.0.0.1` 访问失败。

## 检查

```bash
npm run check
npm test
```

## 当前产品边界

- 进度保存在浏览器 `localStorage`。
- 不包含后端、账号、云同步或内容管理系统。
- 推荐使用本地服务访问，不建议直接打开 `index.html` 文件。
